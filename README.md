## Background

I've spent my career building production ML and LLM systems across compliance, biomedical data, and enterprise software. My work spans the full stack of an AI project — from data pipelines and model training to prompt engineering, eval loops, and agentic workflows.

At **Quinnox**, I joined as the first AI Engineer on a backend Python team taking a compliance platform PoC to production for banks and regulators. I owned the full LLM and evaluation layer: prompt versioning, LLM-as-judge eval with automatic retriggering, a curated golden set for regression testing, a human annotation pipeline, and citation/excerpt extraction for auditable outputs. I also built a solo AI-native PM platform on the Anthropic Claude Agent SDK — connecting Jira, Confluence, and GitHub into a single agent-backed interface — presented to the Quinnox CEO and COOs and Change Managers across UK banks.

Now at **Qyrus** (a Quinnox subsidiary), I'm the first Forward Deployed Engineer — embedded across enterprise QA clients to drive adoption, surface friction, and build POCs including agentic customer onboarding demos for banking clients using LLM, RAG, and MCP evaluation.

Before that, at **Elucidata**, I led and deployed a data validation pipeline processing 1.2M biomedical datasets (GEO, TCGA, PubMed, DepMap), achieving a 32× speedup (8h → 15min) and authored a reusable Python validation library adopted company-wide. At **EY**, I engineered Python ETL for Fortune 500 clients (ADP, IBM) and built Scrapy crawlers that eliminated 20h/week of manual compliance monitoring across 15 jurisdictions.
